# Power-BI-Food-Delivery-Dashboard
An interactive Power BI dashboard designed to analyze food delivery operations, providing insights into orders, revenue, customer behavior, delivery performance, and key business KPIs. The dashboard enables data-driven decision-making through dynamic visualizations and actionable analytics.
